# Lesson 1: Hello World

Welcome to your first lesson! This is a simple introduction to get you started.

## Hello World Exercise

Let's start with a classic "Hello World" example.

### Step 1: Open the Terminal

1. Access your lab environment
2. Open the terminal or command line

### Step 2: Print Hello World

Run the following command:

```bash
echo "Hello, World!"
```

You should see the output: `Hello, World!`

### Step 3: Try a Few More Examples

```bash
# Print your name
echo "Hello, my name is $(whoami)"

# Print the current date
date

# List files in the current directory
ls -la
```

## What You Learned

- How to use basic terminal commands
- How to print text to the console
- Basic command syntax
